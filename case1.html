Realcommercial.com.au is Australia's leader in commercial property experiences. Realcommercial delivers web and native app experiences to support their users property journeys.

Realcommercial is apart of the REA Group a leader in digital property experiences. REA has 20+ leading brands through Australia, Asia and the USA.

Work

As a Senior UI / UX Designer I led the design of responsive web experiences. Realcommercial has a small design team that work within cross functional agile teams. Design followed an adapted HCD process to work in harmony with the wider business. I worked across the full design process from strategy to monitoring user analytics.

REA Group has over 40 designers working across the brands. The wider REA design team of 40 work together to develop process efficiencies and design systems.

Projects

Here are a selection of projects where I was the lead designer:

Property Search & Filtering

Search is the core of the users property journey on realcommercial.com.au. Search allows users to discover properties that meet their needs.

Realcommercial redesigned search & filtering to bring a holistic search experience to users. I was responsible for the responsive web experience. I collaborated with the native app designer on the iOS and Android app redesign.

Property Search Results

The first response to a users property needs. Results allow for comparison to discard or view more details about a property.

This project is the foundation of the realcommercial redesign across web and native. The project started an atomic design system for use on future projects. I was responsible for the responsive web experience redesign.

Property Details

Property details allow for an in-depth look into a potential property before connecting with an agent.

I was responsible for the responsive web experience redesign.

Property Investment

Realcommercial created an Australian first tailored experience for commercial property investment. The experience surfaced unique data points for property investors.

I was responsible for the responsive web experience redesign.
